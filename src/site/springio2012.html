Thymeleaf at the Spring I/O 2012

Spring I/O 2012 Madrid

  • Organizers: SpringSource and javaHispano.
  • Where: Escuela Politécnica Superior of the CEU San Pablo University, Madrid, Spain.
  • When: February 16th and 17th, 2012
  • Event website: http://www.springio.net

The slots you cannot miss

  • Talk: Natural templating in Spring MVC with Thymeleaf
    • Friday Feb 17th, 11:40 - 12:40 - Track 2 (Spring)
    • By José Miguel Samper and Daniel Fernández
    • Talk in Spanish. Slides in English. Will be accepting and answering questions in both languages.
  • Workshop: Learning natural templating in Spring MVC with Thymeleaf
    • Friday Feb 17th, 15:00 - 17:00 - Track 3 (Workshops)
    • By José Miguel Samper and Daniel Fernández
    • Conducted in Spanish. Materials in English. Will be accepting questions and answering comments in both languages.

Workshop requirements and downloads

Title: Learning natural templating in Spring MVC with Thymeleaf

Conducted by means of the Thymeleaf Interactive Tutorial especially developed for Spring I/O 2012. This tutorial uses Thymeleaf version 2.0.1.

Requirements: You need to bring a laptop with some software installed:

  • Java 1.5 or newer
  • A Java application server or servlet container (Tomcat is the preferred option)
  • The text/code editor of your choice, preferably with Java and HTML syntax highlight. Notepad++, TextPad or gedit are fine examples.

Preferably: please download the Thymeleaf Interactive Tutorial [SourceForge.net] beforehand, in order to avoid any possible connection issues during the event. It is just a .war file that can be deployed into your web server.

(Additionally, you can also checkout the interactive tutorial source code from the thymeleafexamples-itutorial module at the SVN trunk.
