thyme :: home

The Thyme Template Engine is work in progress. Publishing of its first stable version is expected in March 2011.

PROJECT STATUS:

  • thyme (core library): Feature-complete and in beta stage.
  • thyme-springmvc3 (Spring MVC integration package): Alpha-stage. Recently finished complete form integration, working on integration of i18n infrastructure. Once done, Thyme will be a full-featured substitute for JSPs in Spring MVC apps.

For the moment everything is quite ugly over here: ultra-simple web, scarcely formatted docs, etc. But much of the software works already! We will soon create a nice website, format our docs, package our binaries and release. If you want to be up-to-date on the project status, follow us on twitter (@thymetemplates).
